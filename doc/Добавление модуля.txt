Если нужно добавить модуль:
1. Добавить в файл conf/modules данные:
		'footer2'	=>
			[	
				'link'		=>	'layouts/footer2',
				'class'		=>	'false',
				'file'		=>	'true'
			]
	где 
		footer2		- 	имя модуля и переменной котору нужно будет вставить в общем ключевом слое
		link		-	ссылка на файл шаблок (находится в каталоге core/class/views/layouts/)
		class		-	класс в папке core/class/mod/modCLASSNAME Где класснейм будет footer2 (так же как и название модуля) при выставлении данного параметра
						ищет этот клас и запускаем метод run() (который должен выставить свойсво App::gi()->$modules['чего то там'] после этого чего то там выставится в отображении )
		file		-	может быть только один (если он установлен то является слоем по умолчанию который загружается в последний момент и отрисовывает все переменные)

2.	В случае необходимости содается класс с методом run() в папке core/class/base/ где чего то там обрабатывается
	class Footer2 extends Singleton{

3.	Создается отображение в папке core/class/views/layouts/
